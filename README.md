The purpose of this repo is to deliver **layered, reusable, and GitHub-friendly** network architecture diagrams for Cloud Solution Architects to run effective Azure design and skilling sessions. The content is based on **real customer and partner design sessions** with collaboration from cross-functional architects. The design areas include Azure Networking, Hybrid connectivity, routing, firewalling, load balancing, multi-region, secure design, and AKS networking. Networking complexity is broken down into **layers** with **one diagram per design area** using [diagrams.net](https://www.diagrams.net/).

---


# Scope: Reusable and Layered Network Diagrams

### Target Audience
Cloud Solution Architects · Network Architects · Cloud Infrastructure Architects · Solution Engineers

### Goals
- Build reusable, GitHub-friendly network architecture diagram templates
- Layered diagrams to run effective Azure design and skilling sessions
- Real-world use cases derived from working with Microsoft customers
- Level 100 → Level 400 scenarios in one diagram
- Understand the *why* behind each design decision
- Minimize lab time with configuration snippets

# Tooling — draw.io (now diagrams.net)
## Features

draw.io is [feature-rich](https://www.diagrams.net/blog). Top 10 features for network architecture diagrams:

1. **[Add Azure Shapes](https://www.diagrams.net/blog/azure-diagrams)** — View → Shapes → Add Shapes → Azure
2. **[Layers](https://www.diagrams.net/blog/interactive-diagram-layers)** — View → Layers → show/hide per scenario
3. **[Scratchpad](https://www.diagrams.net/blog/azure-diagrams)** — View → Scratchpad
4. **Outline** — View → Outline (great for large diagrams)
5. **Flow Animation** — Select flow → Flow animation
6. **[Sketch Style](https://www.diagrams.net/blog/rough-style)** — hand-drawn look for workshops
7. **[Whiteboard Mode](https://www.diagrams.net/blog/change-editor-mode)** — Extras → Theme → Sketch
8. **[Curved Connectors](https://drawio-app.com/curved-connectors-in-draw-io-diagrams/)** — for cleaner flow lines
9. **[VS Code Integration](https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio)** — edit diagrams in VS Code
10. **[Group Shapes](https://drawio-app.com/more-draw-io-shortcuts-to-streamline-your-diagramming/)** — Ctrl+G

## Installation

Three options to open draw.io diagrams:

**Option 1 — Desktop App**

Download from the Microsoft Store, then open any `.drawio` file locally.

![desktop-app](extras/drawio-app.png)

Download a diagram from GitHub: File → Raw → Save link as.

![save-drawio](extras/save-drawio.png)

**Option 2 — Browser**

Open online at [app.diagrams.net](https://app.diagrams.net/)

**Option 3 — VS Code Extension**

![vscode extension](extras/vscode-extension.png)
